PlasmoFP

PlasmoFP Explorer is a web application for exploring protein function prediction for Plasmodium species. Predictions are generated by PlasmoFP, a structure-informed deep learning model developed specifically for Plasmodium species. For more information on PlasmoFP, please refer to the manuscript: PlasmoFP: leveraging deep learning to predict protein function of uncharacterized proteins across the malaria parasite genus.

Harsh R. Srivastava, Daniel Berenberg, Omar Qassab, Ziyi Wang, Richard Bonneau, Jane M. Carlton

Developed by the Jane Carlton lab at Johns Hopkins University Malaria Research Institute.
